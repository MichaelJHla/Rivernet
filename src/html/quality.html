Submit Quality Data

To add data, add the collector, enterer and analyst’s name to the perspective spots. Names only need to be entered once for multiple jars. Then select the correct jar number from the dropdown menu, then pick what data field you want to enter. Once entered, submit the data (this can be edited later.). Not every space needs to be filled out when data is submitted. Once the data is submitted, you can leave the page.
